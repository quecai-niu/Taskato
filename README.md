# Taskato

番茄钟 + 任务管理的 Windows 桌面应用，基于 .NET 8 + WPF，MVVM 架构。

## 功能

### 番茄钟
- 工作/休息计时循环，支持暂停、跳过、提前休息
- 多组独立番茄钟（选项卡切换），每组可自定义命名
- 可调时长：工作 1~240 分钟，休息 1~30 分钟
- 实时进度条、状态动画、托盘同步显示剩余时间

### 任务管理
- 添加任务，支持 4 级优先级（无/中/高/紧急），Tab 键循环切换
- `#标签` 自动解析，标签以药丸样式展示
- 复选框切换完成状态，悬停删除按钮
- 双击任务卡片进入详情编辑，支持修改标题、优先级、完成状态
- 顶部统计今日完成数

### 通知
- 番茄钟完成时屏幕右下角弹出 Toast 窗口，番茄图标弹跳动画
- 提示音：6 种方案（无声 / Notify / Ding / Background / Chimes / 自定义音效），双通道增益播放
- 飞书 Webhook 通知，可分别开启工作完成/休息完成通知

### 历史查询
- 按日期范围和关键词搜索历史任务
- 搜索结果显示完成状态徽章、创建/完成时间
- 支持编辑、删除历史任务
- 导出搜索结果到 CSV（UTF-8 with BOM）

### 其他
- 系统托盘常驻，最小化到托盘后台运行，仅托盘退出才关闭应用
- 6 套渐变主题色（紫/粉/青/橙/绿/蓝），实时切换
- 开机自启（注册表 HKCU Run，无需管理员权限）
- 单例运行，重复启动自动退出
- SQLite 本地存储，异步操作


## 技术栈

.NET 8 · WPF · MVVM · SQLite (sqlite-net-pcl) · Windows Forms (托盘)
