# 番茄钟报错修复与持久化逻辑补全

本方案旨在修复 `InverseBooleanToVisibilityConverter` 缺失导致的报错，补全番茄钟时长保存逻辑，并优化提示音与 UI 细节。

- [ ] **[Sound]** 更换为更响亮的 `Windows Notify.wav`，并优化播放逻辑以解决卡顿。
- [ ] **[UI]** 提升 `ToastWindow` 计时器字体的辨识度。

## 方案细节

### 1. 转换器补全 (Converters.cs & App.xaml)
- 报错原因：XAML 中使用了该资源但未定义。
- 解决：新增转换器类并全局注册。

### 2. 持久化同步 (MainViewModel.cs)
- 在主页点击 `＋/－` 按钮时，同步更新 `_settingsService.Config.WorkMinutes`。
- 调用 `_settingsService.SaveConfig()` 确保下次启动生效。

### 3. 音效优化 (ToastWindow.xaml.cs & notify.ps1)
- 改用 `Windows Notify.wav`（比 Generic 版更响亮、更长）。
- 播放前检查文件是否存在，采用非阻塞方式。

### 4. UI 增强 (ToastWindow.xaml)
- 计时器文本 `ElapsedTimerText`：`FontSize="14"`, `FontWeight="SemiBold"`, `Foreground="#BBBBBB"`。
